# Item Checkout
This project is linking a bootstrap cart and checkout together with some extra functionally. 
- dynamically update total when quantity changes
- delete items from the list
- send total to checkout screen

### Skills:
- React
- HTML5
- CSS3
- Javascript
- Bootstrap
